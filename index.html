Multi-objective Uncapacitated Facility Location Problem with Customers' Preferences: Pareto-based and Weighted Sum GA-based Approaches

Soumen Atta1, Priya Ranjan Sinha Mahapatra1, Anirban Mukhopadhyay1,☆

The instances for the Multi-objective Uncapacitated Facility Location Problem with Customers' Preferences (MOUFLPCP) mentioned in this page are adapted from the Uncapacitated Facility Location Problem (UFLP) [1, 2] and the original instances of UFLP can be found from Beasley's OR Library [3]. Each instance of MOUFLPCP consists of two matrices, viz., service cost matrix and preference matrix, and a set defining the opening cost of each facility.

Format of the data files:
Each instance mouflpcp# of size n × m consists of two data files, namely, "cap#.txt" and "cap#pref.txt". The size of the file "cap#.txt" is n × (m+1) and it consists of service cost matrix of size n × m and the last column of it represents the opening cost. The size of the file "cap#pref.txt" is n × m and it consists of preference matrix.
